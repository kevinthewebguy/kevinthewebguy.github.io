Arduino Robot

Arduino robot that performs 4 functionalities. Firstly, it can detect and follow a line. Secondly, it can be driven by remote control using infrared communication. Thirdly, it can detect objects in front of it and turn 90-degrees to avoid it. Lastly, it can map its surrounding objects using a navigation algorithm.

  • Technologies Used: Arduino, Processing
